# go-mongox
`go-mongox` 是一个基于 **泛型** 的库，对 `MongoDB` 的官方框架进行了扩展，实现了链式调用功能，使得文档操作更加流畅。它还包括多样的 `bson` 构造器和内置函数，以便简化和加速 bson 数据的构建过程。

# 功能特性
- 文档的 `CRUD` 操作
- 聚合操作
- 内置基本的 `Model` 结构体，自动化更新默认的 `field` 字段
- 支持 `bson` 数据的构建
- Hooks
- 支持插件化编程
